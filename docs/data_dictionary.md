# Diccionario de variables

| Variable | Unidad | Descripcion breve |
|---|---|---|
| Date | yyyy-mm-dd | Fecha de medicion |
| Time | HH:MM:SS | Hora de medicion |
| UA, UB, UC | V | Tension fase A/B/C |
| UAvg | V | Promedio de tension de fases |
| IA, IB, IC | A | Corriente fase A/B/C |
| IN | A | Corriente de neutro |
| IAvg | A | Promedio de corriente de fases |
| UTHA, UTHB, UTHC | % | THD de tension por fase |
| UTHAvg | % | THD de tension promedio |
| ITHA, ITHB, ITHC | % | THD de corriente por fase |
| ITHAvg | % | THD de corriente promedio |
| FAvg | Hz | Frecuencia promedio |
| PFAvg | - | Factor de potencia promedio |
| PA, PB, PC | W | Potencia activa por fase |
| PSum | W | Potencia activa total |
| QA, QB, QC | Var | Potencia reactiva por fase |
| QSum | Var | Potencia reactiva total |
| SA, SB, SC | Va | Potencia aparente por fase |
| SSum | Va | Potencia aparente total |
